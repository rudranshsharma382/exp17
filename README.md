# Data Visualization using Matplotlib and Seaborn

## AIM
To study different data visualization techniques using Matplotlib and Seaborn including line charts, bar charts, histograms, and scatter plots.

---

## THEORY

### 1. pd.DataFrame()
Creates a structured dataset for visualization.

### 2. display()
Displays the dataset in tabular format.

### 3. plt.plot()
Creates a line chart to show trends over time or categories.

### 4. marker
Adds symbols (like dots or squares) at data points in line charts.

### 5. plt.xlabel() / plt.ylabel()
Labels the x-axis and y-axis respectively.

### 6. plt.title()
Sets the title of the graph.

### 7. plt.show()
Displays the plotted graph.

### 8. plt.figure(figsize)
Sets the size of the graph (width and height).

### 9. color
Specifies color of plot elements.

### 10. label
Adds legend labels for multiple plots.

### 11. plt.legend()
Displays legend in the graph.

### 12. plt.bar()
Creates bar charts for comparison of values.

### 13. edgecolor
Adds border color to bars.

### 14. plt.grid()
Adds grid lines for better readability.

### 15. bar.get_height()
Retrieves height of each bar in bar chart.

### 16. plt.text()
Adds text labels on graph elements.

### 17. np.arange()
Generates evenly spaced numeric values for plotting positions.

### 18. plt.xticks()
Sets custom labels on x-axis.

### 19. plt.hist()
Creates histogram to show frequency distribution.

### 20. bins
Defines number of intervals in histogram.

### 21. np.mean()
Calculates mean value for reference line in histogram.

### 22. plt.axvline()
Draws a vertical line (e.g., mean line) on graph.

### 23. plt.scatter()
Creates scatter plot to show relationship between variables.

### 24. List Comprehension (colors)
Assigns colors dynamically based on conditions.

### 25. sns.lineplot()
Creates advanced line plots using Seaborn.

### 26. sns.barplot()
Creates bar plots with statistical aggregation.

### 27. sns.histplot()
Creates histogram using Seaborn with better styling.

### 28. sns.scatterplot()
Creates scatter plots using Seaborn.

### 29. errorbar=None
Removes error bars from Seaborn plots.

### 30. Data Visualization
Helps represent data graphically for better understanding and analysis.
